        1:Bucket sort is NOT a comparision based sorting Algorithm.

        3:It has many versions and mostly implemented according to data it is going to receive.
         
        4:In some implementations we even keep the buckets sorted.
       
        5:Its Complexity may vary a lot according to what type of inputs it is receiving.

          its performance.
